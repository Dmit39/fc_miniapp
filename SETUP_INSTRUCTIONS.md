# Farcaster Profile Stats App - Setup Guide

## Быстрый старт

Чтобы приложение работало, нужно добавить NEYNAR_API_KEY:

### Шаг 1: Получите бесплатный API ключ
1. Перейдите на https://neynar.com
2. Зарегистрируйтесь (можно через Gmail)
3. Скопируйте ваш API ключ из dashboard

### Шаг 2: Добавьте ключ в приложение
1. В левой колонке чата найдите кнопку **Vars** (переменные окружения)
2. Нажмите на неё
3. Нажмите "Add Variable"
4. Введите:
   - Key: `NEYNAR_API_KEY`
   - Value: (вставьте скопированный ключ)
5. Нажмите Save

### Шаг 3: Перезагрузите приложение
После добавления ключа приложение должно работать.

## Как использовать

1. В приложении введите ник пользователя Farcaster (например: `vitalik`, `dwr`, `balajis`)
2. Нажмите кнопку "View Stats"
3. Увидите статистику профиля: подписчики, подписки, PFP, FID и дату создания

## Если не работает

- Проверьте что ключ правильно скопирован
- Проверьте что после добавления ключа приложение перезагрузилось
- Попробуйте известного пользователя типа `vitalik` или `dwr`
